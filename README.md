# Viの使用方法  

1.　Viの起動方法  
ターミナルで

 ```$ vi```  
 もしくは  
 ```$ vi ファイル名```  
2. コマンドモード  
立ち上げ時にデフォルトで設定されている  
 文字を入力するには入力モードへの切り替えが必要
* 移動  

| |操作|結果|  
|:---|:---|:---|  
|一文字の移動|h|左|  
  ||j|下|  
  ||k|上|  
  ||l|右|  
|一単語の移動|w|次の単語の先頭|
||b|現在または前の単語の先頭|
||e|現在または次の単語の末尾|

文字、単語の移動には数字と組み合わせて一気に移動が可能
ex)「5w」　5つ先の単語の先頭に移動
